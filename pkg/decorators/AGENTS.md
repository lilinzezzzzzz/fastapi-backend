# AGENTS.md

适用于 `pkg/decorators/`。父级约束见 `pkg/AGENTS.md`。

## 模块职责

通用装饰器集合。当前提供 `async_generator_timer` 等与业务无关的能力装饰器。

## 编码约定

- 保持对 `pkg/logger`、`internal/` 的零硬依赖：
  - 默认 logger 使用标准库 `logging`；如需项目日志，通过装饰器参数注入。
  - 不要在模块顶层 import `internal/`、`pkg.logger`、`pkg.toolkit.context` 等具体业务/基础设施。
- 装饰器参数保留 keyword-only 以便扩展，不要用位置参数堆默认值。
- 装饰器必须保留原函数签名语义：
  - `@wraps(func)` 是硬性要求。
  - 对异步生成器/协程/普通函数的版本拆开提供，避免一个装饰器同时适配多种调用模式造成隐式行为。
- 计时、慢查询阈值等数值通过参数暴露，禁止在模块里写死。

## 验证重点

- 同步函数、协程、异步生成器三类目标函数至少各覆盖一个用例。
- 慢阈值、异常传播、生成器提前终止等边界路径要覆盖。
